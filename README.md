# Netflix Content Analysis Dashboard

Interactive Power BI dashboard analyzing Netflix movies and TV shows.

## Dashboard Preview

![Dashboard Preview](./Screenshot%202026-05-10%20110142.png)

## Tools Used
- Power BI
- Data Visualization
- CSV Dataset

## Key Features
- Movies vs TV Shows analysis
- Top genres analysis
- Top countries analysis
- Ratings distribution
- Content trend over time
- Interactive slicer for content type
